#盲水印工具

一个简单易用的盲水印（Easy Blind Watermark） 命令行/交互式工具，支持"文本水印"和"图片水印"的嵌入与提取。  
基于优秀的 [blind_watermark](https://github.com/guofei9987/blind_watermark) 库实现，无需原始图像即可提取水印，适用于版权保护、图像溯源等场景。

## ✨ 特性

- ✅ 文本水印：将文字信息不可见地嵌入图片中。
- ✅ 图片水印：将一张小图片（如 Logo）嵌入到另一张图片中。
- ✅ 抗攻击：在一定程度上抵抗裁剪、压缩、旋转等常见图像处理（裁剪后图片需大于 384x384 像素)
- ✅ 自动记忆：自动保存水印长度/尺寸及密码，提取时无需重复输入。
- ✅ 密码保护：可自定义密码，提高安全性。
- ✅ 交互菜单：启动后通过数字选择功能，拖拽图片即可操作，简单友好。


首先你得已经成功安装了 https://github.com/guofei9987/blind_watermark 不然无法使用
 一切就绪
根据菜单选择功能

嵌入文本水印 → 拖入原始图片 → 输入文字 → 得到带水印的 PNG 图片。

提取文本水印 → 拖入带水印图片 → 自动或手动输入水印长度 → 显示提取的文字。

嵌入/提取图片水印步骤类似。

💡 所有自动生成的文件默认保存在当前目录，提取时需要的水印信息会自动保存在 last_text_wm.txt 或 last_img_wm.txt 中。

📌 注意事项
输出格式：为保证水印鲁棒性，工具强制输出为 PNG 格式（无损压缩）。

抗裁剪能力：裁剪后的图片尺寸需大于约 384x384 像素，否则水印可能无法提取。

翻拍/截图：通过屏幕翻拍或社交软件二次压缩后，水印极大概率会被破坏，请务必保存原始带水印的 PNG 文件。

密码一致性：嵌入和提取必须使用完全相同的 password_img 和 password_wm。

自定义密码
程序默认使用 password_img=1， password_wm=1。
在每次嵌入或提取时，会询问是否使用默认密码，输入 n 即可自定义。
